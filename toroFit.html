This project it's a fitness web application destined to advisers that do online advises. It facilitates the communication between clients and advisors, and improve the organization and efficiency of the adviser. It search to motivate the clients with stadistics and data of their progres. It's only avaible in Catalan

Front-end

I designed this website with an easy and intuitive user interface, it's full responsive. I've used the HTML and CSS for styling some parts of the web, with the help of bootstrap for styling and doing it full responsive. I've used AJAX in some forms to pass data and receive a response with a notification (toaster), it was done by passing data between front-end and back-end with JSON objects, all of that was with the help of Jquery

Used Technologys

Bootstrap, Jquery, AJAX, JSON, font-awesome, toastrJS

Back-end

The main tecnology of this whole application in Laravel that it's a PHP framework. I've used MySql a database that works very good with laravel. the paypal api is used for doing the payments of the application between the clients and their advisers

Used Tecnologys

PHP, MySql, Laravel, PayPal API

What the application has

Register and login for users clients and advisers

In this application we can differentiate between two different types of users, one who is the advisor and the other who is the client, therefore we will have to create a type of register where we can register client users and advisors and be able to differentiate them
